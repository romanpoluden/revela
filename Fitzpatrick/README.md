## Images — NOT in this repo

The image folder (`finalfitz17k/`) is excluded from Git.  
It contains 16,577 clinical skin images (~3–8GB).

To get the images:
1. Visit https://github.com/mattgroh/fitzpatrick17k
2. Request dataset access
3. Place images in `Fitzpatrick/data/finalfitz17k/`
